# Проекты

Сюда Claude сохраняет результаты работы: коммерческие предложения, письма, посты, анализы.

Каждый файл Claude назовёт понятно, например:
- `кп-клиент-2026-03-25.md`
- `письмо-партнёру-2026-03-25.md`
- `пост-telegram-тема-2026-03-25.md`

Ты можешь создавать подпапки для разных направлений — Claude подстроится.
