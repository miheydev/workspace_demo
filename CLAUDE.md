## Язык и стиль

- Общение: русский
- Стиль документов: деловой, но без канцелярита
- Обращение: на «ты»

## Как работаем

- Результаты — в файлы, не в чат (краткое резюме в чат, полный текст — в файл)
- Спрашивай перед удалением или перезаписью файлов
- Не создавай файлы «на будущее» — только то, что нужно сейчас
- Если задача большая — сначала покажи план, потом делай
- Перед изменением файла — сначала прочитай его

## Структура рабочего пространства

```
Контекст/       — информация обо мне, компании, стиле общения
Проекты/        — результаты работы (КП, письма, анализы, посты)
База знаний/    — документы о бизнесе (услуги, прайсы, кейсы)
Шаблоны/        — шаблоны документов с плейсхолдерами
Входящие/       — файлы для обработки (кинул → попросил разобрать)
```

## Важные правила

- НЕ удаляй файлы без моего подтверждения
- НЕ выводи конфиденциальные данные в чат
- Читай файл перед изменением — никогда не перезаписывай вслепую
- При создании документов — используй шаблоны из Шаблоны/, если подходящий есть
- Используй контекст из Контекст/about-me.md для персонализации документов
