# AI Workspace — Стартовый набор

Твоё рабочее пространство для работы с Claude Code. Скачал — настроил — работаешь.

## Быстрый старт

### 1. Настрой пространство под себя

Открой Claude Code и скажи:

> Помоги мне настроить рабочее пространство под меня. Задавай вопросы — я буду рассказывать голосом.

Claude задаст вопросы о тебе и бизнесе, а потом заполнит два файла:
- **Контекст/about-me.md** — подробная анкета
- **CLAUDE.md** — главная инструкция, настроенная под тебя

### 2. Проверь, что контекст работает

Спроси Claude:

> Расскажи, чем занимается моя компания и какие задачи я хочу решать с AI

Если ответ по делу — всё настроено!

### 3. Попробуй первую задачу

Скажи Claude что-нибудь из:

> Подготовь коммерческое предложение для клиента [имя]

> Проверь мою бизнес-идею: [описание]

> Проанализируй этот документ по методу светофор

Claude найдёт подходящую команду и выполнит.

## Что умеет Claude в этом пространстве

Просто скажи Claude что тебе нужно — он сам найдёт подходящую команду.

| Задача | Что сказать Claude |
|--------|-------------------|
| Настроить пространство | "Помоги настроить пространство под меня" |
| Написать КП | "Подготовь коммерческое предложение для [клиент]" |
| Написать письмо | "Напиши деловое письмо для [кому]" |
| Написать пост | "Напиши пост для Telegram про [тема]" |
| Проанализировать документ | "Проанализируй этот документ" |
| Обработать встречу | "Вот заметки со встречи, сделай саммари" |
| Исследовать рынок | "Проанализируй рынок [ниша]" |
| Исследовать клиента | "Собери информацию о компании [название]" |
| Идеи для контента | "Придумай 10 идей для постов про [тема]" |
| Исследовать тему | "Проведи глубокое исследование по теме [тема]" |
| Проверить идею | "Проверь мою бизнес-идею: [описание]" |
| Анализ светофор | "Проанализируй [объект] по методу светофор" |

## Как добавить свою команду

1. Создай файл в `.claude/commands/` (например `моя-команда.md`)
2. Опиши что Claude должен делать: контекст, шаги, формат результата
3. Теперь просто скажи Claude задачу — он найдёт и используёт команду

## Советы

- **Говори голосом** — это в 3-5 раз быстрее печати
- **Чем больше контекста — тем лучше.** Загрузи документы о бизнесе в `База знаний/`
- **Результат в файл, не в чат.** Claude сам сохраняет результаты в `Проекты/`
- **Кинь файл во Входящие/** и скажи "посмотри что во Входящих" — Claude обработает
- **Новая тема = новая сессия.** Это помогает Claude сосредоточиться на одной задаче
