# Шаблоны

Шаблоны документов с плейсхолдерами. Claude использует их как основу при создании документов.

Что здесь лежит:
- `коммерческое-предложение.md` — шаблон КП
- `бриф-на-проект.md` — бриф для нового клиента/проекта
- `отчёт-по-встрече.md` — структура отчёта после встречи

Как использовать:
- Скажи Claude: "Подготовь КП для клиента [имя]" — он возьмёт шаблон и заполнит
- Можешь добавлять свои шаблоны — Claude будет их находить и использовать

Как добавить свой шаблон:
- Создай .md файл с плейсхолдерами в формате [НАЗВАНИЕ_ПОЛЯ]
- Claude будет заполнять плейсхолдеры на основе контекста из Контекст/about-me.md
