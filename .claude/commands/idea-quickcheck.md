Быстрая проверка бизнес-идеи за 2 минуты. Три эксперта дают честную оценку.

## Что спросить

**Опиши идею в 2-3 предложениях.** Что за продукт/услуга, для кого, как зарабатывает?

## Три эксперта

Оцени идею с трёх точек зрения:

### Продуктолог
- Кому это реально нужно? Есть ли спрос?
- Насколько сложно сделать MVP?
- Что может стать "убийственной фичей"?
- Вердикт: GO / MAYBE / STOP

### Финансист
- На чём конкретно зарабатываем?
- Юнит-экономика: сходится ли математика?
- Сколько нужно вложить до первых денег?
- Вердикт: GO / MAYBE / STOP

### Скептик
- Почему это ещё никто не сделал? (или сделал?)
- Главный риск, который убьёт идею
- Что должно быть правдой, чтобы это взлетело?
- Вердикт: GO / MAYBE / STOP

## Формат вывода (в чат, без файла)

```
ИДЕЯ: [краткое описание]

ПРОДУКТОЛОГ: [вердикт]
[2-3 предложения]

ФИНАНСИСТ: [вердикт]
[2-3 предложения]

СКЕПТИК: [вердикт]
[2-3 предложения]

ИТОГО: [X из 3 — GO]
Главный риск: [одно предложение]
Следующий шаг: [что сделать, чтобы проверить идею]
```

## Важно

- Это быстрая проверка, не бизнес-план
- Будь честным — лучше отговорить от плохой идеи сейчас, чем потом
- Если идея сырая — скажи что доработать, а не просто "плохо"
- Результат выводи прямо в чат, файл не нужен
