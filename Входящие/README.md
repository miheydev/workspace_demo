# Входящие

Папка для быстрой обработки файлов. Кинь сюда любой файл и скажи Claude:

> Посмотри что лежит во Входящих и обработай

Claude прочитает файл, определит что с ним делать и предложит:
- Проанализировать и сохранить результат в Проекты/
- Извлечь ключевую информацию
- Переместить в Базу знаний/
- Ответить на письмо или документ

После обработки файл перемещается из этой папки в нужное место.
