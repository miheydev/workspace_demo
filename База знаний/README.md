# База знаний

Положи сюда документы о своём бизнесе — Claude будет использовать их как контекст.

Что сюда добавить:
- Описание продуктов и услуг
- Прайс-листы
- Презентации компании
- Кейсы и отзывы клиентов
- Внутренние регламенты
- Исследования рынка

Форматы: .md, .txt, .pdf, .docx — Claude прочитает любой.

Чем больше здесь материалов — тем точнее Claude понимает твой бизнес и тем полезнее его ответы.
